# Loan Approval Prediction System

A machine learning application that predicts whether a loan application is likely to be approved based on applicant details. The project compares multiple classification algorithms, selects the best-performing model, and provides real-time predictions through an interactive web interface.

## Features

- Data preprocessing and feature engineering
- Handling missing values and categorical variables
- Multiple ML models:
  - Logistic Regression
  - Decision Tree
  - Random Forest
  - Gradient Boosting
- Model evaluation using cross-validation
- Interactive Gradio interface for instant predictions
- Saved trained model for future inference

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Gradio
